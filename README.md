# Employee Attrition Prediction Model

### Project Overview: This project focuses on building a high-precision machine learning model to identify employees at risk of leaving an organization. By utilizing   predictive analytics, HR departments can implement proactive retention strategies.

### Technical Stack:
Languages:Python (Pandas, NumPy)
Models: CatBoost, Random Forest
Optimization:Hyperparameter tuning via GridSearchCV
Metrics:Optimized for ROC AUC and Recall to minimize false negatives.

### Key Features:
Automated Preprocessing: Handling missing values and categorical encoding.
Ensemble Learning: Comparative analysis between Random Forest and CatBoost.
Performance: Detailed classification reports and feature importance visualizations.
